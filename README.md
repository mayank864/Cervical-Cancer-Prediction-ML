# Cervical Cancer Risk Prediction System

A full-stack ML-powered web application for cervical cancer risk prediction.

## Tech Stack
- **ML**: Python, scikit-learn, Random Forest Classifier
- **Backend**: FastAPI, Uvicorn
- **Frontend**: React, Vite, Tailwind CSS v3

## Project Structure
```
├── ml_research/     # ML training pipeline & model artifacts
├── backend/         # FastAPI prediction API
└── frontend/        # React + Vite frontend
```

## Local Development

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
